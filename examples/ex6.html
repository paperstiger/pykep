from PyKEP import *

This example demonstrates the use of the mr_lt_nep class in the trajopt module to design a hopping mission between main belt asteroids. The class derives from PyGMO problem.base and represents the optimization of a multiple randezvous mission of a low-thrust spacecraft. The initial guess is taken from one of the trajectory submitted by the ACT/ISAS team to the 7th edition of the global trajectory optimisation competition. In essence it tries to hop between four different asteroids of the main belt.

The code for this example can be studied here. Feel free to leave comments.
